⚡

Parallel, not sequential

All models receive your prompt at the same moment. Responses stream in as each one finishes — no waiting for slower models to block faster ones.

🔑

Keys stay in your browser

API keys are stored only in localStorage and sent directly to each provider. The Switchboard server never sees or logs them.

🧵

Per-model conversation threads

Every line keeps its own history. Switch a model mid-conversation, or mute a line without losing context — then flip it back on and continue.

🎙️

Voice input

Click the mic, speak your prompt, review and confirm. Works wherever your browser supports the Web Speech API.

📋

Copy in one click

Copy any response as plain text or Markdown — with the model name embedded — ready to paste anywhere.

💾

Session history

Conversations are auto-saved locally. Resume any past session, and the board picks up exactly where you left off.
